# Customer Segmentation using K-Means

# Overview
This project performs customer segmentation using K-Means clustering based on:
- Purchase Frequency
- Brand Loyalty
- Product Usage

# Features
- Data preprocessing pipeline (imputation + scaling)
- Optimal cluster selection using Elbow Method & Silhouette Score
- 3D visualization of clusters
- Dynamic cluster interpretation

# Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

# How to Run
```bash
pip install -r requirements.txt
python main.py
